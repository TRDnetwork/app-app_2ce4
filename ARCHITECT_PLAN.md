
# Project: one shop test3

## Tech Stack

**Framework:** static-html
**State:** none
**Styling:** vanilla-css
**Data:** none (hardcoded)
**Auth:** none

## Database Schema

No database — static content only.

## API Endpoints

No backend — static site only.

## Frontend Components / Pages

- index.html (landing/homepage with featured products)
- products.html (product listing with category filtering)
- product-detail.html (individual product page)
- cart.html (shopping cart with localStorage persistence)
- checkout.html (checkout flow with order summary)
- about.html (about the store)
- contact.html (contact form — UI-only)
- faq.html (FAQ section)

## Critical Requirements

1. Product grid with "Add to Cart" buttons that persist to localStorage
2. Product detail pages with variant selection (size, color) and image gallery
3. Shopping cart with quantity adjustment, remove item, and total calculation
4. Checkout page with order summary and simulated payment confirmation
5. Mobile-responsive design with hamburger menu navigation
6. Clean, accessible HTML structure with WCAG AA compliance

## Complexity Rating

**Medium** — Multi-page ecommerce site with client-side cart management, category filtering, and responsive design. No backend required; all data hardcoded and cart stored in localStorage.
