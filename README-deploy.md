# Deploy Instructions

## Deploy to Vercel
1. Push code to a GitHub repository.
2. Log in to Vercel and click "Import Project".
3. Select the GitHub repo.
4. Confirm the framework is detected as "Static".
5. Click "Deploy".

## Environment Variables
No environment variables are needed. This is a fully static site.

## First-time setup
No backend setup is required. All product data is hardcoded in HTML files. Cart state is managed via localStorage in the browser.
